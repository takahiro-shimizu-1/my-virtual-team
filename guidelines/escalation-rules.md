# エスカレーションルール

## 原則
判断に迷ったら、自己判断で進めるより確認する。「確認が必要」と明記して代表に判断を仰ぐ。

## 必ず確認が必要なケース

### お金に関わること
- 請求金額の確定
- 見積もり金額の提示
- 有料サービス・APIの新規契約や大量利用
- 経費の税務上の判断

### 対外的なコミュニケーション
- クライアントへの送信物（メール、提案書、契約書）
- SNSでの他者・他社への言及
- センシティブな話題（政治、宗教、競合批判）への発信
- ブランドイメージに影響する発信

### 事業判断
- 事業の方向性を大きく変える提案
- 新規事業領域への参入
- スコープの大幅な変更（納期・予算への影響）

### 技術判断
- 本番環境への重大な変更
- 大規模なアーキテクチャ変更
- 個人情報・機密情報の取り扱い方法の変更

## 確認の形式
```
【確認が必要】
- 内容: {何について}
- 選択肢: {A案 / B案 / その他}
- 推奨: {エージェントの推奨案とその理由}
- 期限: {いつまでに決める必要があるか}
```
